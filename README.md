# Tutly - Мини гид по Уфе

Flutter приложение для исследования достопримечательностей, мест для еды и прогулок в Уфе.

![Flutter](https://img.shields.io/badge/Flutter-3.11+-02569B?style=flat-square&logo=flutter)
![Dart](https://img.shields.io/badge/Dart-3.11+-0175C2?style=flat-square&logo=dart)

## Возможности

- 📱 **Каталог мест** - Просмотр достопримечательностей, кафе, мест для прогулок и фото-спотов
- 🏷️ **Фильтрация по категориям** - Быстрый поиск мест по интересам
- 📍 **Геолокация** - Определение текущего местоположения
- 🗺️ **Режим "Рядом"** - Сортировка мест по расстоянию от пользователя
- 🗺️ **Интерактивная карта** - Просмотр всех мест на карте с маркерами
- 🎨 **Современный дизайн** - Material Design 3 с поддержкой темной темы
- 📐 **Адаптивный интерфейс** - Поддержка iOS, Android, Web, Windows, macOS, Linux

## Скриншоты

### Список мест
- Карточки мест с описанием
- Фильтрация по категориям
- Кнопка "Рядом" для сортировки по расстоянию

### Карта
- Интерактивная карта OpenStreetMap
- Маркеры мест с цветовой кодировкой по категориям
- Маркер текущего местоположения
- Линия маршрута от пользователя до выбранного места

## Технологии

- **Flutter** 3.11+ - UI фреймворк
- **Dart** 3.11+ - Язык программирования
- **flutter_map** 7.0+ - Интерактивная карта
- **OpenStreetMap** - Картографические данные
- **geolocator** 12.0+ - Геолокация
- **permission_handler** 11.3+ - Управление разрешениями

## Разработка

### Структура проекта

```
lib/
├── main.dart              # Точка входа в приложение
├── models/                # Модели данных
│   ├── category.dart      # Категория места
│   ├── place.dart         # Место
│   └── route.dart         # Маршрут
├── screens/               # Экраны
│   ├── places_list.dart   # Список мест
│   └── map_screen.dart    # Карта с маркерами
├── widgets/               # Виджеты
│   └── place_card.dart    # Карточка места
└── services/              # Сервисы
    ├── data_service.dart  # Загрузка данных
    └── location_service.dart # Геолокация
```

### Добавление новых мест

Редактируйте файл `assets/data/places.json`:

```json
{
  "id": 1,
  "name": "Название места",
  "categoryId": "sight",
  "lat": 54.7181,
  "lng": 55.9694,
  "description": "Описание места"
}
```

### Категории мест

- `sight` - Достопримечательности
- `food` - Еда
- `walk` - Прогулки
- `photo` - Фото-споты

## Разрешения

Приложение требует следующие разрешения:

- **Геолокация** - Для определения текущего местоположения и сортировки мест по расстоянию

Разрешения запрашиваются при первом использовании функции "Рядом".
